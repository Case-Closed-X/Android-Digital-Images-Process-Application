# Android Digital Images Process Application
 此项目为我的数字图像处理功能APP，于2021.6.14端午节开源第一版。
 
 项目为Android平台原生开发，采用OpenCV库、Java语言。
 
 项目中使用的图像来源于巫师之昆特牌（强烈安利）、名侦探柯南（强烈安利），灵感来源于小米相册中艺术照片功能（强烈安利），如有侵权，请联系我删除。
