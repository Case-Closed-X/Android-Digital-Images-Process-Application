# Android Digital Images Process Application
 此项目为我大二时候做的数字图像处理功能APP，于2021年端午节开源。
 
 项目为Android平台原生开发，采用OpenCV库。
 
 项目中使用的图像来源于巫师之昆特牌（强烈安利）、名侦探柯南（强烈安利），如有侵权，请联系我删除。
